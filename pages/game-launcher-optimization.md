# Game Launcher Optimization

Game launchers can add background load, overlays, auto-updates, startup entries, and notification services. Optimize them without breaking game installs or anti-cheat.

## Universal Steps

- Disable launcher startup with Windows.
- Disable overlays you do not use.
- Turn off video capture, highlights, and social popups.
- Limit auto-updates to times you are not gaming.
- Keep cloud saves enabled if you rely on them.

## Steam

Use [Optimize Steam](#optimize-steam) for Steam-specific settings.

## Epic, EA, Ubisoft, Battle.net

Check:

- Start with Windows.
- Minimize to tray behavior.
- Overlay.
- Cloud saves.
- Download throttling.
- Beta clients.

## Avoid

- Deleting launcher folders manually.
- Disabling services required by anti-cheat.
- Blocking launcher network access globally.
- Using cracked launcher patches.

## Test

After changes, launch one game per launcher and confirm cloud sync, multiplayer, achievements, and controller support still work.
