# Multimedia etc.

This page covers essential, lightweight, and powerful free tools for customization, recording, screenshots, digital art, image viewing, and music playback. These tools are actively maintained, open source where noted.

## Spotify Customization: Spicetify
The gold standard for modifying the official Spotify client.

* **Key Features**: Full themes, extensions, and custom CSS/JS snippets.
* **Marketplace**: Built-in tab for easy browsing and installing community creations.
* **Compatibility**: Compatible with modern Spotify versions (requires re-application after Spotify updates).
* **Maintenance Note**: After a Spotify update, run `spicetify backup apply`. Check the issue tracker if a new Spotify version causes a temporary break.

| Resource | Link |
| --- | --- |
| Official Site | [spicetify.app](https://spicetify.app/) |
| GitHub Repository | [github.com/spicetify/spicetify-cli](https://github.com/spicetify/spicetify-cli) |

## Screen Recording & Streaming: OBS Studio
A professional-grade, free, and open-source tool with a massive plugin ecosystem.

* **High Performance**: Real-time video/audio capturing and mixing.
* **Studio Mode**: Preview changes before they go live.
* **Modern Features**: OBS supports a large plugin ecosystem, hardware encoders, scene collections, and resilient recording workflows.
* **Platform Support**: Windows 10/11, macOS (Metal renderer support), and Linux.

[github.com/obsproject/obs-studio](https://github.com/obsproject/obs-studio)

## Screenshot & Productivity: ShareX
The most feature-packed screenshot utility available for Windows.

* **Capture Options**: Region shots, scrolling capture, and GIF/video recording.
* **Advanced Tools**: OCR text extraction, image annotations, and region highlighting.
* **Technical Note**: Use the official site or repository and keep ShareX updated through the project's supported channels.
* **Workflows**: Fully customizable "After capture" tasks to automate uploading, saving, and editing.

[getsharex.com](https://getsharex.com/)

## Digital Painting: Krita
Professional-grade painting software designed by artists for artists.

* **Advanced Engine**: Features layers, masks, animation support, and a world-class brush engine.
* **File Support**: Non-destructive editing with full PSD compatibility.
* **Maintenance**: Use the official Krita site for downloads and release notes.
* **Licensing**: Fully open source (GNU GPL).

[krita.org](https://krita.org/)

## Fast Image Viewer: IrfanView
A blazing-fast viewer with a tiny footprint, ideal for quick inspections and batch tasks.

* **Efficiency**: Tiny ~8 MB install size with very low resource usage.
* **Batch Processing**: Powerful batch converter and renamer for massive image collections.
* **Format Support**: Supports almost every image, video, and audio format via a modular plugin system.
* **ARM Support**: Now features native 64-bit ARM builds for modern laptops.

[irfanview.com](https://www.irfanview.com/)
