# Important

Read this before using any guide, tool, script, or download link in this documentation. Your safety matters.

## Technical Accountability

StratumLister is a community-driven resource, and we take care to recommend what works. But every system is different - software changes fast, and what works perfectly on one machine might cause issues on another. Here's how to stay safe:

- **Check Your Version**: Make sure a guide matches your OS build and hardware. Windows 11 23H2 behaves differently than 22H2, and AMD processors don't always behave like Intel.
- **Backup First**: Always create a System Restore point or full disk backup before touching system-level settings. This is non-negotiable.
- **Read Before You Run**: Never run a command or script without understanding what it does. If you don't know what a line of code does, look it up first.

## Link Verification

We take link safety seriously. Here's how we verify what we recommend:

- **Verified Links**: Links with a verification badge have been personally checked by a maintainer. We verified the source and confirmed it's safe to use.
- **Unverified Links**: Links without a badge come from the community. They're useful references, but you should audit them yourself before running anything.

## Who Can Verify?

Only repository maintainers can verify links. Contributors and community members cannot verify their own resources. If someone tells you to skip these checks, that's a red flag - report it to us immediately.

## The Stop & Verify Protocol

Not sure about something? Here's what to do:

1. **Stop** - Don't make any changes yet.
2. **Verify** - Check the official documentation from the vendor (Microsoft, Apple, your GPU manufacturer, etc.).
3. **Report** - Let us know if something seems wrong or outdated. We want to know.
