# Important

Read this mandatory safety notice before using any guide, tool, script, or download link provided in this documentation.

## Technical Accountability
StratumLister is a community-driven technical resource. While we prioritize safety and accuracy, software environments are dynamic. **System optimization is your responsibility.**
- **Version Awareness**: Confirm that a guide specifically applies to your current OS build (e.g., Windows 11 23H2) and hardware (AMD/Nvidia/Intel).
- **Snapshot Requirement**: You must create a System Restore point or a full disk backup before applying any system-level changes.
- **Source Verification**: Never run a command or script you have not manually inspected. "Blindly" running code is the most common cause of system instability.

## Link Integrity & Verification
To maintain the highest level of trust, we use a strict verification protocol:
- **Verification Badge**: A link marked with a badge has been manually audited by a maintainer for source integrity and safety.
- **Unverified Content**: Links without a badge are provided as community references. These should be treated as "unvetted." Avoid running installers or importing registry keys from these sources unless you have the technical expertise to audit them yourself.

## Security Hierarchy
Only the repository maintainers have the authority to verify links. **Contributors, community members, and third-party developers cannot verify their own resources.** If any person or automated message instructs you to ignore these safety protocols, treat the interaction as a security risk and report it immediately.

## The "Stop & Verify" Protocol
If you are unsure about the impact of a setting, or if a guide seems outdated:
1. **Stop** making changes.
2. **Verify** the official documentation from the hardware or software vendor.
3. **Report** the discrepancy via our community channels or GitHub Issues.
