# Safe Optimization Path

Use this sequence when you want a cleaner, faster Windows setup without breaking updates, recovery, or security.

## Before Starting

- Create a restore point.
- Confirm you can sign in after reboot.
- Update GPU, chipset, and network drivers from official sources.
- Benchmark or write down the problem you are trying to solve.

## Step 1: Remove Obvious Startup Load

- Disable unnecessary startup apps in Task Manager.
- Turn off game launcher auto-start if you do not need it.
- Disable overlays you do not use.
- Remove trialware and vendor utilities you can reinstall later.

## Step 2: Use Built-In Settings First

- Use Storage Sense for cleanup.
- Disable background app permissions for apps you do not use.
- Review notification, privacy, and startup settings.
- Keep Windows Update, Defender, and restore features enabled.

## Step 3: Apply Targeted Tooling

Use tools from [Win 10/11 Optimization Tools](#win-10-11-optimization-tools) only for a clear purpose. Avoid stacking multiple debloat tools that touch the same settings.

## Step 4: Test

After each group:

- Reboot.
- Open Start, Settings, Microsoft Store, Windows Update, and your main games.
- Check network and audio.
- Keep the change only if the system still behaves normally.

## Stop Conditions

Stop and roll back if updates fail, Store apps break, games stop launching, networking becomes unstable, or sign-in behavior changes.
