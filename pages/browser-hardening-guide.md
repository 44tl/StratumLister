# Browser Hardening Guide

Browser hardening should improve privacy and reduce malicious ads without breaking normal browsing.

## Baseline Setup

- Keep the browser updated.
- Use a strong content blocker.
- Remove extensions you do not use.
- Disable third-party cookies when practical.
- Use HTTPS-only mode where it does not break required sites.
- Use separate browser profiles for high-trust and low-trust activity.

## Extension Rules

- Install fewer extensions.
- Prefer open-source, actively maintained extensions.
- Avoid extensions that require broad access without a clear reason.
- Remove coupon, shopping, random download, and "video helper" extensions unless you trust the project.

## Privacy Settings

Good defaults:

- Block third-party cookies.
- Clear site permissions you no longer need.
- Disable notification prompts from unknown sites.
- Review camera, microphone, location, and clipboard permissions.

## Do Not Over-Harden

Extreme fingerprinting changes can make you more unique, break sites, or train users to disable protections. A stable browser with a good content blocker is better than a fragile profile full of conflicting tweaks.
